<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- Nodejs (14.x or higher)
- yarn

  ```sh
  npm install --global yarn
  ```

### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/viral-sangani/celo-contractkit-experiments
   ```

2. Install NPM packages

   ```sh
   yarn
   ```
